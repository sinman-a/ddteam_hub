## Frontend Core
1. **HTML5 Semantic** - структура, accessibility, meta tags
2. **CSS3 Advanced** - Grid, Flexbox, CSS Variables, backdrop-filter для glassmorphism
3. **Vanilla JavaScript ES6+** - modules, async/await, localStorage, DOM API
4. **Web Animations API** - для плавних spring анімацій як у Apple
5. **Responsive Design** - mobile-first, clamp(), container queries

## Build & Deploy
6. **Vite** - конфігурація, оптимізація бандлу, tree-shaking
7. **Cloudflare Pages** - налаштування білду, _redirects, headers
8. **Git/GitHub** - CI/CD через Cloudflare Pages integration

## UI/UX Specific
9. **Apple HIG** - Human Interface Guidelines, типографіка, spacing 8px grid
10. **Micro-interactions** - hover states, loading states, feedback animations
11. **Canvas/Confetti** - для анімації успіху через canvas-confetti
12. **i18n** - реалізація перемикання мов без перезавантаження
13. **WCAG** - відповідність стандартам доступності

## Performance
14. **Lighthouse Optimization** - lazy loading, code splitting, preloading
15. **CSS Containment** - для оптимізації reflow/repaint при анімаціях

## Data Management
16. **JSON Schema Design** - структура для питань, категорій, локалізації
17. **Fisher-Yates Shuffle** - для рандомізації питань без повторів
